## 3. e-Stat API v3.0 仕様

仕様URL: https://www.e-stat.go.jp/api/api-info/api-spec
HTML版: https://www.e-stat.go.jp/api/api-info/e-stat-manual3-0

### 基本情報
- ベースURL: https://api.e-stat.go.jp/rest/3.0/app/
- 認証: appId必須
- CORS: JSON形式対応
- https/gzip: 対応済み

### 7つのAPI機能
1. getStatsList — 統計表情報取得
2. getMetaInfo — メタ情報取得
3. getStatsData — 統計データ取得 ★主要
4. postDataset — データセット登録
5. refDataset — データセット参照
6. getDataCatalog — データカタログ取得
7. getStatsDatas — 統計データ一括取得
