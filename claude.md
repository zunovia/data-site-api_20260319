# JAPAN STATS プロジェクトログ (claude.md)

## プロジェクト概要
- **目的**: e-Stat（政府統計の総合窓口）のAPIデータを活用した、シネマティックなデータビジュアライゼーションサイトを構築し、GitHub Pagesで公開する
- **デザイン方針**: 映像的・動画的な体験。スクロールドリブンのモーション重視UI
- **開始日**: 2026-03-19

### 2-3. e-Stat API v3.0 仕様（正式URL: https://www.e-stat.go.jp/api/api-info/api-spec）

**ホスト名**: api.e-stat.go.jp
**現行バージョン**: 3.0（2019/7/26〜）
**認証**: appId（アプリケーションID）必須。ユーザー登録→マイページで発行（最大3つ）
**CORS対応**: JSON形式のURL（`/app/json/`）はCORS対応済み → フロントエンドから直接呼び出し可能
**https対応**: 全バージョンで対応済み
**gzip対応**: Accept-Encoding: gzip でレスポンス圧縮可

#### 7つのAPI機能

| # | 機能 | エンドポイント | メソッド | 用途 |
|---|---|---|---|---|
| 1 | 統計表情報取得 | /getStatsList | GET | 統計表の一覧検索 |
| 2 | メタ情報取得 | /getMetaInfo | GET | 統計表IDの分類・項目情報取得 |
| 3 | 統計データ取得 | /getStatsData | GET | 数値データ本体の取得 |
| 4 | データセット登録 | /postDataset | POST | 絞り込み条件の保存 |
| 5 | データセット参照 | /refDataset | GET | 登録済み条件の参照 |
| 6 | データカタログ取得 | /getDataCatalog | GET | Excel/CSV/PDF等全ファイル一覧 |
| 7 | 統計データ一括取得 | /getStatsDatas | POST | 複数テーブル同時取得 |

#### データ形式
- XML: `/app/getStatsData?...`
- JSON: `/app/json/getStatsData?...`
- JSONP: `/app/jsonp/getStatsData?...`
- CSV: `/app/getSimpleStatsData?...`

#### 主要パラメータ（統計データ取得）
| パラメータ | 必須 | 説明 |
|---|---|---|
| appId | ◯ | アプリケーションID |
| statsDataId | △ | 統計表ID（dataSetIdと排他） |
| cdArea | - | 地域コード絞り込み（カンマ区切り100個まで） |
| cdTime | - | 時間軸コード絞り込み |
| cdCat01〜15 | - | 分類事項コード絞り込み |
| limit | - | 取得件数（省略時10万件） |
| startPosition | - | ページネーション開始位置 |
| metaGetFlg | - | メタ情報同時取得（Y/N） |
| lang | - | J:日本語, E:英語 |

#### JSONレスポンス構造
```
GET_STATS_DATA
├── RESULT (STATUS, ERROR_MSG, DATE)
├── PARAMETER (受信パラメータ)
└── STATISTICAL_DATA
    ├── RESULT_INF (TOTAL_NUMBER, FROM_NUMBER, TO_NUMBER)
    ├── TABLE_INF (統計表メタ情報)
    ├── CLASS_INF (分類情報)
    │   └── CLASS_OBJ[] (@id, CLASS[])
    │       └── CLASS (@code, @name, @level)
    └── DATA_INF
        └── VALUE[] (@tab, @cat01, @area, @time, $=数値)
```

### 3層アーキテクチャ
```
[フロントエンドUI] ← DataService ← [EstatAPI (CORS/JSON)]
                       ↓ fallback
                   [StaticData (静的JSON)]
```

### APIクライアント機能
- `EstatAPI.init(appId)` — 初期化
- `EstatAPI.getStatsList(params)` — 統計表検索
- `EstatAPI.getMetaInfo(id)` — メタ情報取得
- `EstatAPI.getStatsData(params)` — データ取得（メインAPI）
- `EstatAPI.getPrefecturePopulation()` — 都道府県人口（ショートカット）
- `EstatAPI.getAgingRate()` — 高齢化率（ショートカット）
- `EstatAPI.searchStats(keyword)` — 検索（ショートカット）

### DataService機能
- API疎通テスト → 自動フォールバック
- 結果キャッシュ
- データ正規化（APIレスポンス → 統一フォーマット）
